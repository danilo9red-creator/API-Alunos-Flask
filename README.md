**API CRUD com Flask**

Esta é uma API desenvolvida com Python e Flask para realizar operações CRUD (Create, Read, Update, Delete) sobre alunos.

Os dados são armazenados em um arquivo JSON, funcionando como um banco de dados simples.

As rotas podem ser testadas utilizando Postman através da URL: http://127.0.0.1:5000/alunos

**Como executar o projeto**

Clonar o repositório:

Acessar a pasta;

Instalar as dependências;

Executar o projeto;

aplicativo python.py

**Estrutura do projeto**

- app.py - código principal da API;
- dados.json - base de dados;
- README.md - documentação.

**Pontos finais da API**

**Criar/POST** - rota para cadastrar um novo registro na aplicação.
**Listar/GET** - rota para visualizar os registros existentes.
**Atualizar/PUT** - rota para atualizar informações de um registro existente.
**Deletar/DELETE** - rota para remover um registro.
